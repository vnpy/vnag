## 概述

VNAG 是一个模块化、易扩展的 AI Agent 开发框架，主要包含以下核心模块：

| 模块 | 说明 |
|------|------|
| **Agent** | 智能体核心，负责对话管理和任务执行 |
| **Gateway** | 网关层，统一对接各种大模型 API |
| **Tool** | 工具系统，包括本地工具和 MCP 远程工具 |
| **RAG** | 检索增强生成，包括分段器、嵌入器和向量库 |
| **UI** | 图形界面，提供开箱即用的聊天应用 |
